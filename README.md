# AI Фармацевт / Медицинский ассистент

Современное мультимодальное веб-приложение на базе ИИ для помощи в анализе медикаментов, рецептов и симптомов. 

## Архитектура

Приложение построено с использованием **Clean Architecture** и полностью совместимо с развертыванием на **Vercel**.

### Технологический стек
- **Frontend**: React 19, Vite, TypeScript, TailwindCSS 4, Framer Motion.
- **Backend**: Vercel Serverless Functions (/api), OpenAI SDK.
- **AI**: GPT-4o (текст и зрение), Whisper (голос).

### Структура проекта
- `/api`: Серверные функции для обработки чата и голоса.
- `/src/components`: Повторно используемые UI компоненты.
- `/src/store`: Глобальное состояние (Zustand).
- `/src/services`: Интеграция с OpenAI и логика подсказок.

## Возможности
- 📸 **Анализ изображений**: Загрузка фото лекарств, упаковок, рецептов или симптомов.
- 🎤 **Голосовые сообщения**: Поддержка транскрипции через OpenAI Whisper.
- 💬 **Интеллектуальный чат**: Структурированные ответы с оценкой рисков и рекомендациями.
- 💊 **Безопасность**: Автоматические предупреждения и проверка критических состояний.
- 📱 **Mobile-First**: Оптимизировано для сенсорных экранов и мобильных устройств.

## Настройка
1. Скопируйте `.env.example` в `.env`.
2. Добавьте ваш `OPENAI_API_KEY`.
3. Запустите `npm run dev` для локальной разработки.

## Развертывание на Vercel
Приложение готово к деплою простым импортом репозитория. Вся логика API находится в папке `api/`, что автоматически распознается Vercel.
